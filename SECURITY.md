## Reporting a Vulnerability

Please report security issues privately.

- Email: security@cascade.fyi

Please include:
- Description of the issue
- Steps to reproduce
- Impact assessment
- Suggested fix (optional)

## Disclosure Process

1. We acknowledge receipt within 48 hours
2. We assess severity and scope
3. We prepare and validate a fix
4. We coordinate disclosure timing with the reporter

## Safe Harbor

We will not pursue legal action against researchers who act in good faith, avoid privacy violations, and report issues privately before public disclosure.
